El Bollo El bollo de pescado consiste en una base de plátano verde, pescado y maní envuelta en una hoja de plátano. Este delicioso plato se lo puede disfrutar con una bebida, puede ser limonada, jugo de naranja, entre otros. Además de una porción de arroz.
